# GameWatcherBot
게임 시작하면 알림 오는 봇 ~~일명 게임 스토킹 봇~~

## How to Use
1. git clone
2. install package
3. edit settings.example.json to settings.json 

### settings.json
``` json
"bot_token": "discord bot's token",
"channel_id": "discord channel id to send message",
"hypixel": {
    "key": "hypixel api key",
    "player_uuid": "minecraft user's uuid"
}
```
